# FF14MSQThematicTracker
A simple ff14 tracker designed to help new players experience the ff14 story in a thematic way.

AI usage and reuse disclosure: I used AI to source information and create this tracker. The prompting that went into the research is biased based on how I consumed the game and might not fit how you want to experience the MSQ. If you decide to use this, please understand that I made this on a whim for one device usage, so when you use this, it will not save across devices. The intent is to keep it local and isolated for personal progression use. I will not be offering cross-device or in-game modding support. I simply don't have the time for that. Anyone is free to use this tracker to build such functionality, I only request that you pay it forward in that you disclose the AI usage.
