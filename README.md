# FF14MSQThematicTracker
A simple ff14 tracker designed to help new players experience the ff14 story in a thematic way.



AI usage and reuse disclosure: AI was used to source information and create this tracker. The prompting that went into the research is biased; based on how I consumed the game. This might not fit how you want to experience the MSQ. If you decide to use this, please understand that I made this on a whim for one device, personal use. It will not save across devices. 

Again: The intent is to keep it local and isolated for personal use. I will not be offering cross-device or in-game modding support. Anyone is free to use this tracker to build such functionality, I only request that you pay it forward in that you disclose the AI usage.
